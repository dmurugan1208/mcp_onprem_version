ISSUE 2: Token Expiration in Long Test Runs
  Tests: Multiple (CONC101, WORKFLOW100, VALIDATE100/101, LIST100/101, etc.)
  Severity: HIGH
  Status: CONFIRMED (affects ~30% of failures)
  Description: JWT expires after ~10+ sequential operations
  Impact: Tests fail with 401 instead of proper error
  Root Cause: JWT has 7-day TTL but tests run sequentially
  Recommendation:
    - Implement token refresh in test suite, OR
    - Extend JWT TTL for testing, OR
    - Run test groups in parallel
  Fix Priority: HIGH (for test reliability)

ISSUE 3: Response Format Inconsistency
  Tests: RESP101, RESP102, RESP103, RESP104, LIST100, LIST101

  Status: CONFIRMED

  Impact: API contract violation - breaks client code
  Root Cause: Unclear (potential race condition or format issue)
  Recommendation:
    - Standardize response format
    - Always return arrays for list operations
    - Review list endpoint implementation
  Fix Priority: HIGH

  Recommendation: Investigate synchronization in create operation
  Fix Priority: MEDIUM
